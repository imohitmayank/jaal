ğŸ‘‰ FeaturesÂ¶

At present, the dashboard consist of the following sections,

  1. Settings panel: here we have multiple options to play with the graph data. It further contains the following sub-sections,

    1. Search: can be used to highlight a node in the graph

    2. Filter: supports pandas query language and can be used to filter the graph data based on nodes or edge features.

    3. Color: can be used to color nodes or edges based on their categorical features. Note, currently only features with at max 20 cardinality are supported.

  2. Graph: the network graph plotted using visdcc.
